#### 多级标题

"#" 

快捷键 ctrl + 1-6 六级标题

ctrl + 0 清空多级标题

#### 有序列表

"1."

快捷键 ctrl + shift + [

降级快捷键 tab

升级快捷键 shift + tab

#### 无序列表

"*" 或 "-"

快捷键 ctrl + shift + ]

#### 任务列表

"-" + 空格 + "[" + 空格 + "]" 

#### 插入表格

"|"分隔

ctrl + enter 创建下一行

快捷键 ctrl + t

#### 行内代码

"`"隔开

快捷键 ctrl + shift + `

#### 代码块

"~~~" 或 "```" + 语言名称

快捷键 ctrl + shift + k

#### 插入图片

"!" + "[]" + "()" 

快捷键 ctrl + shift + i
